# Invoxa

A professional invoicing app for freelancers and small businesses, built with Next.js and Tailwind CSS.

## Features

- Create professional invoices with client details
- Add line items with quantities and rates
- Auto-calculated totals
- PDF download (coming soon)

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The dev server runs on **http://localhost:5000**

### Build

```bash
npm run build
npm start
```

## Tech Stack

- [Next.js 14](https://nextjs.org/) — React framework (App Router)
- [Tailwind CSS](https://tailwindcss.com/) — Utility-first CSS
- TypeScript

## Project Structure

```
app/
  layout.tsx    # Root layout
  page.tsx      # Invoice form (/)
  globals.css   # Global styles
```
